# Mypage Portfolio

Đây là trang portfolio thiết kế với giao diện hiện đại, hiệu ứng động và hình ảnh phong cách.

## Cách xem

1. Mở `index.html` trong trình duyệt.
2. Hoặc dùng Live Server nếu có sẵn trong VS Code.

## Nội dung

- `index.html` - cấu trúc trang
- `styles.css` - kiểu dáng và animation
- `script.js` - hiệu ứng xuất hiện khi cuộn
